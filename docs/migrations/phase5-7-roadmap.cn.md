- [ ] 多用户数据完全隔离（无跨用户泄漏）

- [ ] Token 用量准确（误差 ≤ 5%）
- [ ] 测试覆盖率 ≥ 80%

### Phase 6

- [ ] 简历定制 API 生成 Markdown (30 秒内)

- [ ] 测试覆盖率 ≥ 80%

- [ ] 混合检索在评估数据集上召回率提升 ≥ 10%

- [ ] 个人词汇权重提升可量化
- [ ] 测试覆盖率 ≥ 80%
